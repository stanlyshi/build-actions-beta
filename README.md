### 说明

---

OpenWrt用作lxc容器部署时，会有部分兼容性问题，做了以下补丁，目前完美运行。

lxc版本OpenWrt部分补丁：

-  autocore.patch——修改CPU信息显示
-  ethinfo.patch——修复ethtool: bad command line argument(s) For more information run ethtool -h错误
-  index.patch——修复最大连接数无法获取而显示默认4096
-  sysctl.patch——网络优化
